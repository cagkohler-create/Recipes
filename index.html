Instructions
Appear in Instagram's share menu
step 1
Host these 3 files on any free HTTPS host — drag the recipenote/ folder onto Netlify Drop, Vercel, or GitHub Pages. You'll get a free yourname.netlify.app URL.
step 2
Open the URL in Chrome on Android (or Safari on iOS). After a moment, Chrome will show an "Add to Home Screen" banner, or tap the ⋮ menu → Add to Home Screen. On iOS Safari, tap the Share icon → Add to Home Screen.
step 3
On Android: once installed, open any Instagram post, tap Share → More, and Recipe·Note will appear in the list. Tap it and the recipe extracts automatically. 🎉

On iOS: the Share Target API isn't supported in Safari PWAs yet. Use the URL paste mode or copy the caption instead.
